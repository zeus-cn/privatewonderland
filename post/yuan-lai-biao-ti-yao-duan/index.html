原来标题要短

2019-05-21

今天看到帖子里面的评论功能不能用，我就纳闷儿了，之前还能用呢，然后就打开了自己的静态博客，发现有的评论就没问题，嘉那篇帖子的评论就有问题。

赶紧打开Gitalk Document，看了看里面的要求，无意中发现了：

id String

Default: location.href.

页面的唯一标识。长度必须小于50。

懵懵懂懂的我觉得可能和评论不能用有关系，所以就去修改了每篇博客的标题。

我发现页面的唯一标识就是博客的标题拼音

结果当然是成功了！（目前已经在设置中将页面标识换成Short ID了）
